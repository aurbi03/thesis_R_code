# Music Recommendation Label Experiment

This repository contains the R analysis code and pseudonymised outputs for my master's thesis on whether recommendation labels influence music evaluations.

## Contents

- `thesis_analysis.R`: Main R script used for data cleaning, descriptive statistics, reliability checks, mixed-effects models, moderation analyses, and robustness checks.
- `DBI Master Thesis - Music Streaming Platforms and Music Evaluation_June 29, 2026_02.22.csv`: Pseudonymised dataset used for the analysis.
- `analysis_output`: Tables and model outputs reported in the thesis.

## Reproducibility

To reproduce the analyses, open `thesis_analysis.R` in RStudio and run the script from top to bottom.
